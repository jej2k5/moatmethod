# Moat Method

This repository hosts a growing collection of **SKILLs** and supporting resources for GTM work based on the **Moat Method™** methodology.

## What you'll find here

- GTM-focused SKILLs for repeatable execution
- Supporting playbooks, templates, and reference materials
- Practical guidance aligned to the Moat Method™ approach

## Purpose

The goal of this repo is to make Moat Method™ GTM practices easier to adopt, operationalize, and scale through reusable assets.

## Skills

### Moat Diagnostics
Runs interactive GTM diagnostics for deep tech founders, covering:
- Narrative Health
- Competitive Displacement
- Deal Health
- POV Strength

Each diagnostic outputs a score, diagnosis, key signals, and 2-3 actionable next moves.

[Open the Moat Diagnostics skill](./moat-diagnostics.skill)

### Signaly

Guided lead research that finds and scores prospects using custom signal criteria. Industry-agnostic. Walks you through defining your ICP and scoring signals, then researches and ranks matches. Built for outbound.

[Open the Signaly skill](./signally.skill)

### Sales Wargame

Pit your sales team against a named competitor inside a real deal. Round-by-round simulation where both sides make moves, a neutral buyer judges each round, and a randomized disruption hits mid-game. Ends with win probability and the moves that actually decided the outcome.

[Open the Sales Wargame skill](./sales-wargame.skill)
