# Moat Method

This repository hosts a growing collection of **SKILLs** and supporting resources for GTM work based on the **Moat Method™** methodology.

## What you'll find here

- GTM-focused SKILLs for repeatable execution
- Supporting playbooks, templates, and reference materials
- Practical guidance aligned to the Moat Method™ approach

## Purpose

The goal of this repo is to make Moat Method™ GTM practices easier to adopt, operationalize, and scale through reusable assets.
